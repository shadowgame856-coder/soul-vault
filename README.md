這是一個融合「超自然現象」與「數位終端機美學」的個人展示專案。透過 Web 技術模擬一個地府與現實交界的系統介面，記錄著處於「系統異常」狀態下的超現實作業環境。

---

## 💻 技術堆疊 (Tech Stack)
本作品集透過以下技術建構出濃厚的「黑客/終端機」氛圍：
- **HTML5/CSS3**: 實作 CRT 掃描線特效（Scanline）、故障藝術（Glitch Art）及打字機特效。
- **JavaScript (ES6+)**: 
  - **因果修補程序**: 模擬系統決策邏輯與隨機事件跳轉（50% 機率跳轉至 404 頁面）。
  - **遺言解碼工具**: 摩斯密碼（Morse Code）加解密演算。
- **SweetAlert2**: 整合深色模式的彈窗互動，提升系統化的視覺反饋。
